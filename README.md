# FlyWise 

FlyWise is a full-stack airline management and flight booking system
developed as a team First-Year Academic Project.

This repository contains the **frontend part** of the application,
which I was responsible for designing and implementing.

## My Role
- UI/UX design
- Frontend development using HTML, CSS, and JavaScript
- Integration with backend APIs (during the project)

## Project Features
- Flight search and booking
- Destination management
- Responsive airline website design

## Tech Stack
Frontend:
- HTML5
- CSS3
- JavaScript

Backend (team project):
- PHP
- SQL database

## Note
The backend code is not included in this repository as it was developed
by other team members as part of the same project.
